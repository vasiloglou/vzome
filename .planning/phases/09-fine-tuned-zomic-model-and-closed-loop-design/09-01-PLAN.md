# Phase 9 Plan 01

## Objective

Define the typed acceptance/eval-set artifact layer that later model
improvements will use.

## Tasks

1. Add typed Phase 9 schema models for eval sets, acceptance packs, and their
   summaries/manifests.
2. Implement deterministic helpers that export an eval set from existing corpus
   artifacts and build an acceptance pack from benchmark metrics.
3. Add focused schema/helper coverage for the new artifact family.

## Acceptance criteria

- A typed eval-set/export path exists on disk.
- A typed acceptance-pack artifact exists on disk.
- The new contracts are additive to the existing LLM runtime surface.
