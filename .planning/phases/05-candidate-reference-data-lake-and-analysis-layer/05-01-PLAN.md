Create `materials-discovery/src/materials_discovery/lake/catalog.py`:
- Define Pydantic model `CatalogEntry` with fields:
  - `artifact_type: str` (one of: "source_snapshot", "reference_pack", "processed", "candidates", "screened", "hifi_validated", "ranked", "active_learning", "prototypes", "calibration", "benchmarks", "reports", "manifests", "execution_cache", "registry", "external_fixtures", "external_pinned")
  - `directory_path: str` (workspace-relative, per D-04 and review concern #5)
  - `schema_version: str` (read from artifact files if present, else "unknown")
  - `record_count: int` (count JSONL lines for JSONL files, count JSON files for JSON dirs)
  - `last_modified_utc: str` (ISO8601, from newest file mtime in the directory)
  - `lineage: dict[str, str]` (workspace-relative pointers to run config, reference-pack manifest, source snapshot, per review concern #5)
  - `size_bytes: int` (total size of directory contents)
  - `is_stale: bool` (from staleness check)
  - `content_hash: str | None` (SHA256 of sorted output_hashes from manifest if present)
- Define Pydantic model `DirectoryCatalog` with fields:
  - `schema_version: str` = "catalog/v1"
  - `generated_at_utc: str`
  - `entries: list[CatalogEntry]`
- Implement `build_directory_catalog(directory: Path, artifact_type: str) -> DirectoryCatalog`:
  - Scan directory for artifact files (*.jsonl, *.json)
  - Count records (JSONL lines for .jsonl, file count for .json)
  - Sum file sizes
  - Extract lineage from manifest files if present (using workspace_relative for all paths)
  - Extract schema_version from first JSON/JSONL file's "schema_version" field if present
  - Compute content_hash from manifest output_hashes if a manifest is found
  - Call check_staleness to set is_stale
- Implement `write_catalog(catalog: DirectoryCatalog, directory: Path) -> Path`:
  - Writes `_catalog.json` into the given directory
  - Returns the path written

Create `materials-discovery/src/materials_discovery/lake/staleness.py`:
- Implement `check_staleness(directory: Path, catalog_path: Path | None) -> bool`:
  - If catalog_path is None or does not exist, return True (no catalog = stale)
  - Compare directory's newest file mtime against catalog file mtime. If directory is newer, return True. (Timestamps as hints per review suggestion.)
  - If a manifest JSON exists in the directory, load it and compare its `output_hashes` against the `content_hash` recorded in the existing catalog entry. If they differ, return True. (Hash-based staleness per review suggestion.)
  - Otherwise return False.
