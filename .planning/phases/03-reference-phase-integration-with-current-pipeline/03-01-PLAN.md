# Phase 3 Plan 01: Build The Reference-Phase Projection Core

## Objective

- **What:** Add the canonical-source projection layer that converts Phase 2
  staged records into the existing processed reference-phase contract.
- **Why:** The source runtime already stages canonical records, but the current
  pipeline still expects `data/processed/{slug}_reference_phases.jsonl`.
- **Output:** A deterministic projection module plus focused tests that lock the
  bridge semantics before the CLI is wired to use it.
