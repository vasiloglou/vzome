# Phase 8 Plan 02

## Objective

Integrate LLM assessments into downstream reporting and ranking context without
changing the core rank score formula.

## Tasks

1. Add additive assessment metrics helpers for Phase 8 report/benchmark use.
2. Thread LLM assessment blocks through report evidence and summary output when
   `llm-evaluate` artifacts are present.
3. Extend ranking/report regressions to prove backward compatibility and the new
   additive context path.

## Acceptance criteria

- Reports surface assessment context when available.
- Ranked/report artifacts remain valid when no LLM evaluation has run.
- No hidden reweighting of `hifi-rank` scores occurs in Phase 8.
