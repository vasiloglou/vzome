# Roadmap: Materials Design Program

**Milestone:** `v1.3`
**Focus:** Project 3 expansion - Zomic-Native Local Checkpoint MVP
**Numbering mode:** continue after `v1.2` (`Phase 25+`)

## Milestone Summary

This milestone extended the shipped `v1.2` serving workflow into the first real
Zomic-adapted local checkpoint lane.

`v1.3` delivered:

- file-backed checkpoint registration and lineage
- one adapted local checkpoint lane inside the shipped generation workflow
- replay-safe checkpoint fingerprint identity
- adapted-vs-baseline benchmark comparison on a shared acceptance-pack context
- operator docs for registration, smoke testing, rollback, and comparison

## Phase Roadmap

### Phase 25: Zomic Checkpoint Artifact and Lineage Contracts

**Primary requirements:** `LLM-19`, `OPS-11`

Shipped the typed registration contract, registration CLI, deterministic
checkpoint storage, and additive serving identity for adapted checkpoints.

### Phase 26: Zomic-Adapted Local Generation Integration

**Primary requirements:** `LLM-20`, `LLM-21`

**Primary requirements:** `LLM-22`, `OPS-12`

runbook coverage for registration, smoke testing, rollback, and benchmark
interpretation.
