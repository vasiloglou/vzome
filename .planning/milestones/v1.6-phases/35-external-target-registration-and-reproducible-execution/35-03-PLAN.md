Add three commands to `materials-discovery/src/materials_discovery/cli.py`:

1. `@app.command("llm-register-external-target")`
   - accepts `--spec PATH`
   - calls `register_external_target(spec_path, root=workspace_root())`
   - prints the returned `LlmExternalTargetRegistrationSummary` as JSON

2. `@app.command("llm-inspect-external-target")`
   - accepts `--model-id ID`
   - loads the registration artifact and, when present, the environment and smoke artifacts
   - prints a concise human-readable trace of immutable target identity plus latest reproducibility artifacts

3. `@app.command("llm-smoke-external-target")`
   - accepts `--model-id ID`
   - calls `smoke_external_target(model_id, root=workspace_root())`
   - prints the returned `LlmExternalTargetSmokeCheck` as JSON

Keep the command surface narrow:
- do not add comparative benchmark execution
