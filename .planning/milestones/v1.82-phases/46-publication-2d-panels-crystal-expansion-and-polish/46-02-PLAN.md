   - Generate offsets: `itertools.product(range(n_cells), repeat=3)` -> 8 combinations for n_cells=2
   - Create ONE trace per orbit (5 traces total, NOT 40):
     - For each orbit, iterate all 8 offsets x all sites
     - Coordinate: `x = (fp[0] + n1) * a - mc_cart[0]`, same for y, z
     - Build opacity list: 1.0 if (n1, n2, n3) == (0, 0, 0), else 0.3
     - Build hover text: `f"{site['label']} -- {shell_name} -- {'central' if is_central else f'cell ({n1},{n2},{n3})'}"` (use -- not em-dash for ASCII safety)
   - Add `go.Scatter3d(x=xs, y=ys, z=zs, mode="markers", marker=dict(size=5, color=color, opacity=opacities), name=shell_name, text=texts, hovertemplate="%{text}<extra></extra>")`
   - Layout title: "Sc-Zn Periodic Approximant -- 2x2x2 Expansion" (or f"{n_cells}x{n_cells}x{n_cells}")
   - Scene axes: xaxis_title="x (A)", yaxis_title="y (A)", zaxis_title="z (A)", aspectmode="data"
   - Return fig (do NOT call fig.show())
