# Phase 8 Plan 01

## Objective

Add the first `llm-evaluate` contracts, runtime engine, and CLI command.

## Tasks

1. Add additive `LlmEvaluateConfig` and `LlmEvaluateSummary` contracts plus
   typed Phase 8 schema models for requests, assessments, and run manifests.
2. Implement `llm/evaluate.py` to read ranked candidates, build prompts,
   resolve the existing LLM adapter seam, write `data/llm_evaluated/` JSONL,
   and persist a run manifest.
3. Add `mdisc llm-evaluate` to `cli.py` and add focused schema/CLI tests.

## Acceptance criteria

- `mdisc llm-evaluate` exists and prints JSON summary.
- Ranked candidate input is the default evaluation seam.
- Outputs remain additive and do not fork `CandidateRecord`.
