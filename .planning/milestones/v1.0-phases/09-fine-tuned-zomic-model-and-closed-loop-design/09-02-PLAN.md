# Phase 9 Plan 02

## Objective

Strengthen `llm-generate` with optional composition-conditioned example packs.

## Tasks

1. Add an additive config seam for example-pack/eval-set conditioning.
2. Extend prompt assembly and run-manifest lineage so `llm-generate` can inject
   reproducible few-shot examples when configured.
3. Add focused core/CLI regressions proving the conditioned path stays optional
   and keeps standard `CandidateRecord` output.

## Acceptance criteria

- `llm-generate` works unchanged when no example pack is configured.
- Prompts include deterministic conditioning examples when configured.
- Run artifacts preserve which example pack or eval set was used.
