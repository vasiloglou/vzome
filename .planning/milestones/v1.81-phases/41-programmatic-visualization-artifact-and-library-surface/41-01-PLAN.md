<objective>
Ship a tutorial-first raw-export visualization library and one thin CLI wrapper
so the checked Sc-Zn design can be rendered programmatically without asking the
user to open desktop vZome for the happy-path preview.

Purpose: satisfy VIS-01 and VIS-02 without turning Phase 41 into a general
visualization platform or a browser-parity project.
Output: a new `materials_discovery.visualization` package, a `preview-zomic`
command, focused regression coverage, a narrow reference doc, and the required
`materials-discovery/Progress.md` update.
</objective>

- Handle zero-segment or points-only exports without crashing normalization or bounds calculation.
