# CLAUDE.md

## Project

vZome monorepo including the `materials-discovery/` no-DFT quasicrystal pipeline.

## Progress Tracking

**Every time you make changes to the `materials-discovery/` directory, you MUST update
`materials-discovery/Progress.md`:**

1. **Changelog table** — add a new row with the date, a short change title, and a brief details column.
2. **Diary section** — append a timestamped entry (use the current date heading, e.g. `### 2026-03-23`) describing what was done, why, and any open items. If a heading for today already exists, append under it rather than creating a duplicate.

This applies to all actions: code changes, config changes, experiment runs, bug fixes, refactors, new systems, doc updates, etc. No change is too small to log.
